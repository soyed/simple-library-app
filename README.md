## Overview
### Problem Statement
Create a library application that allows the following operations:
* Rent a book from the library
* Return borrowed books
* Remove | delete books
  * Restricted to Admin and Staff members
* Get all books
* Get all rented books
* Get all staff members
  * Restricted to Admin
